1. Initialization :
	- git init
		
	* adds empty hidden '.git' repository which stores credentials and repo paths.

2. Connect to GitHub repository :
	- git remote add origin [YOUR REPO’s LINK]
	
	* Repo-link  : can be get from repo's top right corner's 'code' button. consider 'HTTPS' link.  

3. Log In :
	- git config user.email "your@email.address"
	- git config user.name "UserName"

4. Add credentials :
	- git config credential.email "your@email.address"
	- git config credential.username "UserName"
	
5. Uploadation :
	Select files :
		- git add [ . | filename | foldername]
		
		* .  : represents all or newly added data.
		* file/folder name :  specific file or folder
	
	Generate commit id :
		- git commit -m "message"

	Push data :
		- git push origin master
